# 都道府県マスター (Prefecture Master)

日本地図を使ったインタラクティブな都道府県学習・クイズアプリです。
ブラウザ上で動く設計で、地方ごとの色分けや直感的なアニメーションを通じて、楽しく都道府県や県庁所在地を覚えることができます。

[**👉 GitHub Pagesで遊ぶ**](https://metrojuice.github.io/PrefectureQuiz/)  

## ✨ 主な機能と特徴

用途に合わせて右上のセレクトボックスからモードを切り替えられます。

* **都道府県クイズ**
  地図から都道府県を選び、その県を当てるモードです。
* **県庁所在地クイズ**
  地図から都道府県を選び、その県の「県庁所在地」を当てるモードです。
* **暗記モード**
  リストに「都道府県名 (県庁所在地)」が併記されます。リストから選ぶか、地図から直接県をタップすると、その場所が明るくハイライトされ、特産品や豆知識が記載された解説カードが表示されます。
